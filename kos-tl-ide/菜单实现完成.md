## 功能说明

### 文件操作
- **新建文件**: 创建新的未命名文件
- **打开文件**: 通过对话框选择并打开文件
- **打开工作区**: 打开目录作为工作区
- **保存**: 保存当前文件
- **另存为**: 将文件保存到新位置

### 编辑操作
- 所有标准编辑操作（撤销、重做、剪切、复制、粘贴、全选）
- 查找和替换（Monaco Editor 内置功能）

### 编译和运行
- **编译**: 编译当前文件为 C 代码
- **类型检查**: 检查当前文件的类型错误

## 使用方法

### 通过菜单
1. 点击菜单栏中的菜单项
2. 选择相应的操作

### 通过快捷键
- 使用菜单中显示的快捷键
- 例如：`Ctrl+N` 新建文件，`F5` 编译

## 技术实现

### 菜单创建
- 文件: `main/menu.ts`
- 使用 Electron Menu API
- 支持 Windows、macOS、Linux

### 事件传递
- 主进程通过 `webContents.send()` 发送菜单事件
- 渲染进程通过 `ipcRenderer.on()` 接收事件
- 通过 preload 脚本安全暴露 API

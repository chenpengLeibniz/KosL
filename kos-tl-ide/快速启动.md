# KOS-TL IDE 快速启动指南

## ✅ 构建状态

**构建已成功完成！** 所有修复已包含在构建中。

## 🚀 立即启动

### 方法1: 开发模式（推荐）

```powershell
cd kos-tl-ide
npm run dev
```

这会自动：
- 编译主进程代码
- 启动 Webpack 开发服务器
- 启动 Electron 应用

### 方法2: 直接运行

```powershell
cd kos-tl-ide
.\node_modules\.bin\electron.cmd dist/main/main.js
```

## 📝 测试步骤

1. **启动 IDE** - 使用上述任一方法

2. **打开文件**:
   - 点击"打开文件"按钮
   - 或按 `Ctrl+O`
   - 选择 `examples/test.kos`

3. **编辑代码**:
   - 在编辑器中输入代码
   - 查看语法高亮
   - 测试代码补全（输入 `def` 或 `type`）

4. **保存文件**:
   - 按 `Ctrl+S`
   - 或点击保存按钮

5. **编译文件**:
   - 按 `F5`
   - 或点击编译按钮
   - 查看编译输出面板

## 🎯 快捷键

- `Ctrl+O` / `Cmd+O` - 打开文件
- `Ctrl+S` / `Cmd+S` - 保存文件
- `Ctrl+P` / `Cmd+P` - 命令面板
- `F5` - 编译

## ⚠️ 注意事项

1. **编译器路径**: 如果编译器未配置，编译时会显示错误提示
2. **首次运行**: 可能需要几秒钟加载 Monaco Editor
3. **网络**: IDE 本身不需要网络，但类型检查需要本地编译器

## 运行方式

### 方式1: 使用已存在的可执行文件（推荐）

如果之前已经打包过，可以直接使用：

```powershell
cd kos-tl-ide\release\win-unpacked
.\KOS-TL IDE.exe
```

### 方式2: 开发模式运行

```powershell
cd kos-tl-ide
npm run dev
```

这会：
1. 编译主进程（watch 模式）
2. 启动 Webpack 开发服务器
3. 启动 Electron 应用

### 方式3: 直接运行 Electron

```powershell
cd kos-tl-ide
.\node_modules\.bin\electron.cmd dist/main/main.js
```

## 注意事项

### 打包问题

当前打包时遇到代码签名工具下载问题，这是网络/权限问题，不影响功能：

- **解决方案1**: 使用已存在的打包文件（如果存在）
- **解决方案2**: 使用开发模式运行
- **解决方案3**: 稍后重试打包（可能需要管理员权限或网络连接）

### 编译器配置

首次使用时，如果编译器未配置：

1. 编译功能会显示错误提示
2. 可以在设置中配置编译器路径
3. 默认路径: `../../compiler/build/bin/kos-tl-compiler`

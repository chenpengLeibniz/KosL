# KOS-TL IDE 菜单栏使用指南

## 菜单栏已实现

IDE 现在拥有完整的菜单栏，包含所有核心功能。

## 快速开始

### 文件操作

1. **新建文件**:
   - 菜单: 文件 → 新建文件
   - 快捷键: `Ctrl+N` / `Cmd+N`

2. **打开文件**:
   - 菜单: 文件 → 打开文件
   - 快捷键: `Ctrl+O` / `Cmd+O`

3. **保存文件**:
   - 菜单: 文件 → 保存
   - 快捷键: `Ctrl+S` / `Cmd+S`

### 编译运行

1. **编译**:
   - 菜单: 运行 → 编译
   - 快捷键: `F5`

2. **类型检查**:
   - 菜单: 运行 → 类型检查
   - 快捷键: `F6`
